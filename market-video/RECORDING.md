# 声クローン用 録音ガイド(一度だけ・10〜15分)

## 手順(スマホで完結)
1. https://elevenlabs.io → Sign up → **Starterプラン($5/月)** に加入(声クローンと商用利用に必要)
2. 左メニュー **Voices → Add a new voice → Instant Voice Clone**
3. スマホのマイクで下の台本を録音してアップロード(合計2〜3分以上。静かな部屋で、マイクから20cm、普段の解説口調で)
4. 作成された声を開き **Voice ID** をコピー
5. プロフィールアイコン → **API Keys → Create API Key** でキーを発行
6. GitHubリポジトリ(ブラウザで「デスクトップ用サイト」表示)→ Settings → Secrets and variables → Actions で2つ登録:
   - `ELEVENLABS_API_KEY` = 発行したキー
   - `ELEVENLABS_VOICE_ID` = コピーしたVoice ID
   → 以後の自動生成が自動的にあなたの声に切り替わります(コード変更不要)

## 録音台本(実際の動画と同じ語彙で読むと品質が上がります)
「おはようございます。今朝の海外マーケットをデータで確認していきます。エスアンドピー500は前日比でプラス、ナスダックも上昇しました。ドル円はやや円高方向です。」
「シカゴの日経平均先物は、前日の終値より高い水準で戻ってきています。ただし、この数字は夜間にすでに織り込まれた結果です。」
「セクター別では、半導体と一般消費財に買いが集まりました。東京市場では関連する銘柄群の動きに注目です。」
「今回の検証では、直近10回分のデータをすべて並べました。結論から言うと、通説とは異なる結果になっています。」
「数字の出典は概要欄にあります。本動画は情報提供であり、投資助言ではありません。投資判断はご自身でお願いします。ご視聴ありがとうございました。」
※ 上を2回、少し速さを変えて読むと約3分になります。読み間違えてもそのまま続けてOK(自然さが大事)
