# 競合分析レポート: フリーランス向け請求・入金管理SaaS(候補A)

調査日: 2026-07-17
調査方法: 3系統の並行調査(①請求書作成サービス大手、②入金管理・回収・資金繰りの隣接領域、③ユーザーの生の声・利用実態)+ 重要ファクト10件の敵対的検証(1クレーム1エージェントで反証を試行)

---

## 1. エグゼクティブサマリー

- 請求書「**発行前**」(作成・送付)は無料ツールが乱立する激戦区、「**換金**」(ファクタリング)もプレイヤー過密。しかし「**発行後**」(期日監視・督促・回収・新法対応)を個人フリーランス価格帯で担うサービスは**ほぼ空白**。
- 特に「**受注側フリーランスのためのフリーランス新法60日ルール自動チェック**」は、反証を試みた検証でも**該当サービスが存在しないことを確認**(発注側=企業向けのみ存在)。最も確度の高い差別化ポイント。
- ただし完全な空白ではない: **Zoho Invoice(無料)が顧客宛て自動支払リマインダーを提供**しており、「自動督促」単体では差別化にならない。勝負所は「日本の商習慣に合った角の立たない督促」+「新法対応」+「税シミュレーション」の組み合わせ。
- **最大の脅威はfreee**。2025年7月にFREENANCE買収を発表し(9月子会社化、10月吸収合併)、請求→即日払い→補償の垂直統合を進行中。回収領域へ拡張する土台を持つ唯一の大手。
- **制度前提の重要アップデート**: インボイス2割特例は予定通り2026年9月30日を含む課税期間で終了するが、**令和8年度税制改正で個人事業者限定の「3割特例」(2027年分・2028年分)が創設済み(成立済みの確定制度)**。「2026年で終わる一発需要」ではなく、**2026〜2029年にかけて毎年制度が変わり続ける**ため、税シミュレーション需要はむしろ長期化する。

### 各社の要点と弱み

- **Misoca(弥生)**: 個人フリーランスの無料受け皿の筆頭(月10通)。見積→請求変換、源泉徴収10.21%自動計算、郵送代行あり。「Misoca回収保証」(売掛金未回収を補填、保証料約0.8%、有料プラン限定)という独自オプションも検証で実在確認。ただし2018年に無料枠を大幅縮小した経緯があり、無料依存ユーザーには「また改悪されるのでは」という警戒が残る。督促・新法・税機能はない。
- **freee**: freee請求書は発行無制限で無料だが、入金消込・申告はfreee会計(年11,760円〜)への誘導構造。「消費税納税額シミュレーション」(無料、4問で一般課税/簡易課税/2割特例を比較)は税務面で最強。解約まわりの不満(「解約させてくれない」等)がみん評に集中しており、ブランドへの不信層が一定数いる。
- **マネーフォワード**: 請求書単体プランがなく月900円〜のセット契約のみ(単体契約は51名以上限定と公式FAQで確認)。「未入金請求書アラート」は毎週火曜に自分宛てメール通知で、回収側では大手の中で最も手厚いが、督促そのものは手動。2024年10月発表の値上げで「確定申告は年1回なのに毎月900円」という不満が顕在化。
- **INVOY**: 無料で発行し放題(事実上)が武器。口座連携(Moneytree)での入金突合はStandard(月980円)限定。請求書カード払い(支払を最大60日延長)など資金繰り機能が特徴。会計・申告・税機能はない。
- **board**: 見積〜受注〜請求〜入金〜売上分析の「業務管理」型で受託業には実務的だが、無料プランがなく単発請求のフリーランスには機能過剰。
- **Zoho Invoice**: 完全無料+顧客宛て自動督促という機能面では最強の反例。ただし①インド資本で日本語UIに機械翻訳感、②郵送代行・国内会計ソフト連携・適格請求書の様式細部など日本の商習慣対応が弱い、③外貨消費税端数処理非対応、という「日本仕様の壁」が実質的な参入余地を残している。

## 4. 隣接競合: 「発行後」の領域

### ファクタリング/報酬即日払い(換金)

| サービス | 手数料 | 特徴 |
|---|---|---|
| FREENANCE(freee傘下) | 3〜10%(初回10%) | あんしん補償(最高5,000万円)無料付帯。**2025年10月にfreeeへ吸収合併済み** |
| ペイトナー | 一律10% | 利用者の9割超が個人。最短10分入金 |
| labol | 一律10% | 24時間365日、最短30分 |

- いずれも請求書1枚単位の「換金」であり、売掛台帳・期日管理・督促は提供しない。手数料10%は恒常利用には高すぎる。

### 入金消込・債権管理SaaS

- 請求管理ロボ(**月5万円〜**)、V-ONEクラウド、Lecto — すべて中堅企業〜エンタープライズ向け。最安のクロジカ請求管理(月4,980円〜)でも中小企業向けで、**個人価格帯(月0〜2,000円)には誰も降りてきていない**。

### フリーランス新法(2024年11月施行)対応

- **発注側(企業向け)は充実**: freee業務委託管理(旧pasture)が3条書面発行・60日期日アラートを実装。Lansmart by SmartHR、電子契約系も対応。
- **受注側(フリーランス自身向け)は検証でも空白を確認**: 存在するのは公取委パンフ、無料テンプレ、手動の支払期日計算ツール(toolkitslab)のみ。「自分の請求の支払期日が60日を超えていないか自動検知・警告」するツールは6方向の検索でも見つからなかった。

### 未払いトラブルの受け皿(事後対応)

- フリーランス・トラブル110番(無料弁護士相談): 相談は月1,000件超に増加、最多は「報酬の支払い」関連(約3割)。
- フリーランス協会の弁護士費用保険(年会費1万円に付帯)、少額訴訟・支払督促(セルフ or 弁護士)。
- **予防と初動(期日超過の検知→角の立たない督促→証拠化→相談導線)をつなぐ中間層が欠落**している。

## 5. ユーザー実態(需要側の裏付け)

- **ツール利用**: 個人事業主の55.6%は会計ソフト自体を未利用(Excel・手書き)。請求書は「無料Excelテンプレのダウンロード」が依然巨大な入口で、ベンダー自身がテンプレ配布→有料誘導をSEO戦略にしている。
- **未払い・遅延**: 報酬の不払い・減額経験は約25%(連合2025)。トラブル時に約6割が事実上の泣き寝入り(内閣官房系調査)。契約は「メール・チャット合意」72.3%、「口約束」56.0%と曖昧。
- **督促の心理的ハードル**(生の声): 「お金に細かいと思われたくない」「催促したら関係が悪くなるかも」「督促時は『経理担当』の人格になりきる」— **感情労働の外部化**に明確な需要。
- **税の不安**: 「2割特例が終わったらどうなる?」「簡易課税の届出はいつまで?」を税理士ブログが量産=検索需要の裏付け。インボイスで価格転嫁できた人は18.7%のみ。税理士依頼(5〜10万円)は高いと感じる層が自力で検索している。
- **支払い意欲の三層構造**: ①完全無料でないと使わない層(過半)、②年5千〜1万円強なら時短対価として払う層、③税理士に5万円以上払う層。ターゲットは②で、価格アンカーはfreee会計(年11,760円)・弥生(年27,800円)・MF(年10,800円)。

### 刺さるペルソナ(要件定義のインプット)

1. **月2〜3枚テンプレ請求のデザイナー/ライター**(30代・年商300〜600万): Misoca無料枠とExcelを行き来。サブスク不信。→「テンプレ並みの気軽さ+インボイス対応+勝手に課金されない」
2. **催促できず入金管理が崩壊しているひとり親方型**(40代・複数取引先): 未払い経験あり、督促文が書けない。→「期日自動監視+失礼にならない督促文の自動生成+新法の根拠添付」に月500〜1,000円
3. **2割特例で申告してきた課税1年生**(売上〜800万): 「特例終了後どうすれば」で検索を繰り返す。→「売上を入れると本則/簡易/3割特例を即比較+届出期限リマインド」に単発課金(1,000〜3,000円)の余地

## 7. 結論: 空白地帯と差別化戦略

### 確認された空白(参入余地)

1. **受注側の新法対応**(60日ルール自動検知・3条書面の記録/証拠化) — 完全な空白。チャット合意72.3%という実態に対し「発注条件の証拠化」から入るワークフローは独自価値
2. **日本仕様の督促自動化** — Zohoという機能的反例はあるが、「源泉徴収・適格請求書・敬語の督促文・経理締め日への配慮・新法を根拠に添える」という日本仕様の組み合わせは空白
3. **請求データと直結した税シミュレーション** — freeeの無料シミュレーターは強いが単発ツール(手入力)。**自分の請求実績から自動計算**され、2026→2027(3割特例開始)→2029(特例消滅)の制度変化を毎年ナビゲートする体験は存在しない

### ポジショニング

> 「請求書を**送ったあと**を守るツール」— 発行は無料ツールに任せてもいい。期日監視・督促・新法・税金という"送ったあとの不安"を月額ワンコインで引き受ける。

- 発行機能も持つが、主戦場を「発行後」に置くことで Misoca/INVOY/Zoho と正面衝突しない
- freee/MF の会計スイート(年1〜3万円)より軽く安く、「会計ソフトはまだ要らない」55.6%層を取る

### 脅威と対策

| 脅威 | 深刻度 | 対策 |
|---|---|---|
| freeeの垂直統合(FREENANCE統合済み、回収領域へ拡張可能) | 高 | 統合が完成する前のスピード勝負。freeeが軽視する「無料〜月980円」帯に先に根を張る |
| Zoho Invoiceの無料自動督促 | 中 | 機能比較でなく「日本の商習慣対応」(源泉徴収・敬語文面・新法根拠・国内会計連携)で差別化 |
| Misocaの無料枠拡大・機能追加 | 中 | 弥生は「発行」に最適化しており回収側の優先度が低い。新法・税は弥生会計側の縄張りで請求書単体には来にくい |
| 3割特例による「2026年9月クリフ」の緩和 | 低 | むしろ好材料: 制度変化が2029年まで毎年続くため、税ナビゲーション需要が長期化 |

- 1年目: 有料100人=月10万円。フリーミアム転換3〜5%前提で無料登録2,000〜3,000人
