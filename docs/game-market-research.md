# ゲーム市場調査レポート: エンタメ転換(野球ゲーム・RPG×アイテム課金)

調査日: 2026-07-17
経緯: 候補A(フリーランス向けSaaS)の競合調査後、方向性を「エンタメ(野球ゲーム・RPG系、アイテム課金)」へ転換。ゲーム向けに市場・法規制・技術を再調査した。
調査方法: 3系統の並行調査(①個人開発ゲームの成功事例と売れるジャンル、②アイテム課金の法規制とマネタイズ設計、③野球ゲーム/RPGの競合とバイブコーディング適性)

---

## 1. エグゼクティブサマリー

- 個人開発ゲームで再現性が高い勝ち筋は **「熱量の高い日本語ニッチ × 育成/放置シミュレーション × 広告+少額課金ハイブリッド」**。野球はその筆頭で、先行事例『私を甲子園に連れてって』が**累計1,290万円・最高月商257万円**を実証済み。
- **公式のスマホ版栄冠ナイン『栄冠クロス』が2026年3月にサービス終了**し、「スマホで手軽に遊べる高校野球監督シム」の本命が不在に。検索需要は恒常的に残っており、明確な空白が生まれている。
- アイテム課金の法規制は設計で回避可能: **ゲーム内通貨を発行せず「アイテム直接販売」にすれば資金決済法をほぼ構造的に回避**できる。ガチャは非推奨(景表法・運用負荷)。
- 技術面では、**戦闘アクションのないテキスト/UIベースのシミュレーションならゲームエンジン不要**でWeb技術(React/Next.js)で作れる。バイブコーディング適性は最高クラス。
- ゲームはSaaSと違い**ヒット依存の分布**(Steamの中央値粗収益は約249ドル、99%は埋もれる)。リテンション設計と「感情に沿った課金設計」が成否を分ける。

---

## 2. 個人開発ゲームの成功事例(数字はすべて出典付き)

### 最重要事例: 『私を甲子園に連れてって』(ふるあぷ/個人開発)

- 高校野球シミュレーション(2017年リリース、iOS/Android)。2018年9月時点で約4.8万DL、**累計収益1,290万円、最高月商257万円**(甲子園シーズンの8月にスパイク)([アプリマーケティング研究所の取材](https://appmarketinglabo.net/furuapp-koushien/))
- 収益比率は**課金76% : 広告24%**。課金ユーザー率5〜6%、課金者単価約3,000円
- 転機は「ドラフト課金」: **引退する3年生を約100円のチケットで対人戦用OBチームに残せる**仕組みで課金収益が10倍化。「思い出の選手を残したい」という感情に沿った少額課金の威力を実証
- 同氏の2作目(草野球あるある育成シム)も累計300万円・月30万円で、「あるあるネタ×育成シム」の再現性を確認([同研究所](https://appmarketinglabo.net/furuapp-baseball/))

### その他のジャンル別事例

### プラットフォーム別の現実

- **スマホ**: 「10万〜50万DLで月20万〜200万円」が相場感。日本語ニッチ(野球・部活)はストア検索で浮上しやすい
- **Steam**: 上限最大だが分布は極端。2025年は発売の40%が登録費100ドル未回収、**中央値粗収益は約249ドル**([AUTOMATON](https://automaton-media.com/articles/newsjp/steam-20251022-362555/))
- **ブラウザ**: ゲームアツマール終了(2023)以降、収益源ではなく**体験版・ファン獲得の導線**と割り切るのが共通認識
- ハイパーカジュアル(純広告型)は縮小し、**広告+課金の「ハイブリッドカジュアル」に移行**(採用率68%→81%)

### 失敗パターン

- 埋もれる(宣伝ゼロ+ストア放流で99.9%はかすらない)/ 翌日離脱50%・週間離脱70%が普通(リテンション設計必須)/ 課金ポイントを後付けにする / 単一アプリ・単一収益源依存で急落

## 3. アイテム課金の法規制(2026年時点・個人開発向け実務)

### 資金決済法(最重要)

- ジェム等の**ゲーム内通貨は「自家型前払式支払手段」に該当し得る**。基準日残高1,000万円超で財務局届出+**残高の1/2以上の供託義務**(個人には致命的)。サ終時の払戻し義務も重い
- 「有効期限6ヶ月未満」なら適用除外だが、**App Storeガイドライン3.1.1が課金通貨への有効期限を禁止**しておりiOSでは使えない
- **推奨解: 通貨を発行せず「アイテム直接販売」のみで構成**(取得時点で役務提供が完了するアイテムは前払式支払手段に非該当)。無償配布のボーナス通貨は対価性がなく非該当

### ガチャ(景品表示法)

- **コンプガチャは違法**(カード合わせ禁止、個人にも適用)。確率表示は法定義務ではなくCESA/JOGA自主規制だが、表示確率と実際の抽選が異なると**有利誤認で措置命令**の実例あり(スクエニ、gumi等)
- **個人開発ではガチャ自体を非推奨**(確率管理・天井・炎上対応の運用負荷が最重量)。やるなら全確率表示+天井+コンプ要素ゼロ

### 未成年課金

- 18歳未満の課金は法定代理人の同意がなければ**民法5条で取消し可能**。あっせん事例の約87%で一部以上返金 → 返金は来る前提で設計
- 実務: 課金前の生年月日確認(警告文付き)+年齢帯別の月間課金上限+課金履歴表示

### 手数料とプラットフォーム選択(スマホ新法後)

- Apple(日本): IAP=21%(Small Business Program 10%)+決済処理5%。外部リンク販売にも15%(SBP 10%)のストアサービス手数料が残り、税務・報告義務は開発者側
- Google: 外部リンク20%(〜$100万は10%)、User Choice Billing 4%減。外部決済は事業者登録必須
- **結論: 年商100万ドル未満の個人はプラットフォームIAP(実質15%程度)が最も安全で手取りも大差ない**。Steamは30%。ブラウザ+Stripe直販は手取り最良だが特商法・返金・不正対策を全部自分で負う

### 特商法・規約

- アイテム課金は通信販売 → **「特定商取引法に基づく表記」必須**。個人の住所・電話は「請求があれば遅滞なく開示」の記載で省略可、バーチャルオフィスも条件付きで容認([消費者庁Q&A](https://www.no-trouble.caa.go.jp/qa/advertising.html))
- 利用規約(払戻し不可・未成年条件・サ終時の扱い・限定免責)、プライバシーポリシー(広告SDK使用時は外部送信規律への対応)をリリース前に整備

## 4. 野球ゲームの競合マップと権利問題

### 競合マップ

| 型 | 代表 | 個人開発から見た位置づけ |
|---|---|---|
| リアルアクション×実名 | プロスピA、パワプロ | 絶対王者。正面から戦わない |
| 育成サクセス | パワプロアプリ | 大手の資本力ありき |
| **監督シム(高校野球)** | **栄冠クロス → 2026年3月サ終** | **公式撤退で空白。狙い目** |
| キャラもの運営型 | ハチナイ → 2024年12月サ終 | 重運営型は個人に不向きの証左 |
| 本格経営シム | OOTP(Steam) | 日本語非対応。ライト層に空白 |
| 個人開発インディー | 私を甲子園に連れてって、甲子園物語、栄冠にゃいん | 好評だが品質不満(運ゲー・UI)が残り、品質面の参入余地 |

- 栄冠クロスの敗因は「ガチャ特待生・スタミナ制・重い周回」= **ユーザーが求めていたのは本家栄冠ナインの「買い切り的で自由に周回できる」体験**。サ終後、知恵袋等で「スマホの代替がない」という声が発生
- 「栄冠ナイン スマホ」「高校野球 シミュレーション アプリ」系の検索需要は2026年も恒常的

### 権利問題(個人開発の前提)

- **NPB球団名・選手実名はライセンス必須**(選手の氏名・肖像は球団が独占的使用許諾権を持つと最高裁で確定)→ **架空チーム・架空選手一択**
- **「甲子園」は阪神電鉄の商標**(野球興行等の区分)。タイトル・ゲーム内呼称での使用はリスクが高く、**「全国大会」等への言い換えが安全**。「高校野球」という一般語・題材自体は問題なし

## 5. 技術スタックとバイブコーディング適性

| スタック | 適性 | 備考 |
|---|---|---|
| **Web技術(React/Next.js)** | ◎ | 戦闘アクションのないテキスト/UIベースの監督シム・放置ゲームならエンジン不要。AIの学習データが最も厚い。Melvor Idleが実例 |
| Phaser(JS) | ○ | 「AIに投げれば動く」と評されバイブコーディング相性良。2Dアクションが必要な場合 |
| Godot | ○ | 全ファイルがテキストでAIエージェント向き。ネイティブ寄りの場合 |
| Unity | △ | バイナリシーン+.metaがAI操作と相性が悪い |

### 1人運営の現実

- 運営型の教訓(IDC2025座談会): 「期間限定イベントをやめて**恒常コンテンツ追加型**にする」「サーバー依存を減らす」が無理なく続けるコツ
- 『私を甲子園に連れてって』型(買い切り的アイテム+広告、ガチャなし、DAU約3,000)は**1人でも回る規模感が実証済み**

## 6. 結論

1. **題材**: 高校野球監督シム(架空チーム・架空選手、「甲子園」の語は回避)が、需要の空白(栄冠クロス撤退)×先行実証(月商257万円)×日本語モートの交差点
2. **形**: テキスト/UIベースのシミュレーション → Web技術で開発しバイブコーディング適性を最大化。モバイルはCapacitor等でラップしてストアIAP(15%)
3. **課金**: 通貨なし・ガチャなしの**アイテム直接販売+リワード広告+広告除去買い切り**のハイブリッド(法規制を構造的に回避しつつ、先行事例の収益構成76:24を踏襲)
4. **タイミング**: 甲子園シーズン(3月センバツ・8月選手権)に収益がスパイクする季節性を狙ってリリース計画を組む

具体的な候補案と評価は [game-candidates.md](./game-candidates.md) を参照。
