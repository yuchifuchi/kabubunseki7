# 調査: 動画投稿自動化システムの自己運用による収益化(2026)

調査日: 2026-07-19 / 方向転換: アプリのリリースではなく「自分専用の動画生成・投稿自動化システム」を構築・運用し、動画プラットフォームで収益化する。

## 1. 結論サマリー

- **成立する。ただし「フル自動」ではなく「半自動」が2026年の勝ち筋**。YouTubeは2025年7月に「量産型(inauthentic)コンテンツ」ポリシーを明確化し、2026年1〜2月に量産系チャンネルの一斉収益化停止が発生(ジャンルの約2割)。一方、**自作台本+独自構成なら合成音声(ずんだもん/ゆっくり)でも明確にセーフ**で、高収益事例が継続中。
- 分水嶺は「自動化しているか」ではなく「**1本ごとに人間の付加価値が見えるか**」。生成・編集・レンダリング・投稿は自動化してよい。**テーマ選定・台本の独自性・ファクトチェック・公開判断だけ人間が握る**。
- 運用コストは**月1,000円前後**(ローカルTTS+ローカルレンダリング+LLM API)で始められる。

## 2. 収益実態(検証済みの数字)

| 事例 | 規模 | 収益 |
|---|---|---|
| ずんだもん国際情勢解説 | 登録1.9万人 | **月124万円** |
| ゆっくり解説(20万人超) | 毎日投稿期 | 月150〜400万円 |
| ゆっくり解説(アニメ考察) | 収益化直後 | 月103〜105万円 |
| ずんだもん歴史解説(小規模) | 登録2,000人 | 月1,500〜1万円(=初期の現実) |

- 長尺RPM(日本): 一般100〜200円/千再生、**ビジネス・金融・IT系は500〜1,000円超**。ShortsはRPM10〜50円で長尺の1/10以下 → **長尺主軸+自分の長尺からの切り出しShorts併走**が正解
- 収益化条件: 登録1,000人+4,000時間(12ヶ月)or Shorts1,000万回(90日)
- 非属人チャンネルはM&A市場で売却可能(出口がある)

## 3. 規制の要点(2026年に守るべきライン)

- ❌ AI台本をそのまま読み上げ/サイト文章の単純読み上げ/テンプレ大量複製/1日3本超の連投/素材・サムネの使い回し多用
- ✅ 自作台本・独自の構成/一次情報・独自データ/週1〜2本ペース/制作実態を示せる(収益化停止時の復活事例あり)
- AI開示: リアル誤認系は開示必須。アニメ調・ゆっくり系は不要。2026年5月からAIラベル自動付与開始

## 4. 技術構成(推奨スタック)

```
ネタ収集(RSS/API) → LLMで台本JSON → [人間: 台本チェック・修正]
  → VOICEVOX(ローカル・無料・要クレジット表記)で音声
  → 音声長から字幕・カット自動算出
  → Remotion(React、個人無料)でテンプレ合成(いらすとや20点以内/自作素材)
  → ローカルレンダリング(FFmpeg)
  → YouTube Data APIで投稿 → [人間: 最終プレビュー+公開判断]
```

- **YouTube API制約**: 監査未通過プロジェクトのAPI投稿はprivateロック(2026年も有効)。ただし**個人でも監査は通せる**(1週間〜数週間、通過事例あり)。監査前は「APIでprivate投稿→Studioで手動公開」、通過後は`publishAt`予約公開
- クォータは2025年12月に緩和され1日約100本まで投稿可能(実用上無制限)
- TikTok APIは未監査だと本人のみ閲覧に強制→当面は書き出し手動投稿。Instagramは自分のアカウントのみなら審査不要で自動投稿可
- 権利: VOICEVOX/ずんだもんは商用OK(クレジット必須、キャラ利用は個人無償)。いらすとやは1動画20点まで。BGMは規約個別確認。「〇〇風」AI画像は回避
- 参考実装: MoneyPrinterTurbo(9万star)、Remotion+LLMのゆっくり解説自動化事例(日本語)、YMM4+ChatGPT半自動化

## 5. コスト・工数

- 金銭: LLM+電気代で**月数百〜1,000円**(ElevenLabs等を足しても月3,000〜6,000円)
- 時間: 手作業なら1本8〜12時間 → 自動化で**1本30分〜2時間(台本チェックと公開判断のみ)**。持続ラインは週1〜2本

## 6. 次の論点(要決定)

1. **テーマ/ジャンル**: 継続の生命線。高RPN狙いなら金融・IT・ビジネス解説、これまでの流れなら野球・スポーツデータ解説、他に時事・雑学・ランキング等。**「独自データ・一次情報を持てるか」で選ぶ**
2. キャラクター: ずんだもん(実績最多・レッドオーシャン気味)vs ゆっくり vs 無キャラ(データビジュアル型)
3. 投稿ペース: 週1〜2本(推奨)からの立ち上げ
4. YouTube API監査: 初日から申請するか(通過まで手動公開運用)

## 7. 追加壁打ち(2026-07-19): データソース・競合・マネタイズ

- **データソース**: SEC EDGAR(13F/決算/インサイダー取引)・FRED(マクロ統計)・CFTC COTはすべて公的無料データで権利問題なし。株価はyfinance/Stooq等の遅延データで十分
- **競合判定: 部分的に空いている**。属人系大手(バフェット太郎50万人等)は飽和だが、「13F・機関投資家データ・決算数値の可視化を主役にした非属人チャンネル」は日本語圏で空白(この領域はトウシル等のテキストメディアのみで動画化されていない)。ゆっくり/ずんだもん投資系は小規模のみ
- **収益構成の推奨**: ①AdSense広告(金融RPMは1再生1円超もあり)→②証券口座アフィリエイト(moomoo約1,250円+取引連動、SBIは資料請求成果。A8はYouTube概要欄掲載を公式サポート。PR表記必須)→③メンバーシップ/Kindle(一般的解説に限定)
- **絶対のレッドライン**: 会員限定・有料での個別銘柄推奨は投資助言業の無登録営業(刑事罰)。「データの見方・一般的解説」に徹する

## 8. 海外バズジャンルの日本移植 有望度ランキング(2026-07-20調査)

1. 経済・金融ドキュメンタリー型アニメ解説(How Money Works型: 170万人・月$1.7万) — 日本は顔出し/静止画中心で空白。金融CPM最高
2. 3D比較・スケール可視化(MetaBallStudios型: 個人で200万人) — 日本語専業不在で空白度最高。言語依存低く輸出可
3. 歴史地図アニメ(Every Day形式) — 体系的大手不在。テンプレ化容易・エバーグリーン
4. バーチャートレース(日本ローカルデータ) — 自動化適性最高だが量産型判定リスク大(独自データ+解説音声必須)
5. アンビエント睡眠×日本的世界観 — 国内競合ありだが輸出型

バズ構造: 「知ってる物の意外なスケール感」「順位の逆転劇」「身近な金の裏側」。完全自動チャンネルの97%は収益化前に脱落 → 半自動+独自データが生存条件(再確認)
